# webpack-hmr

Quickly build a webpack HMR(Hot Module Replacement)

# Usage

``` bash
# serve with hot reload at localhost:8080
npm run dev

# you can custom host and port
npm run dev -- --host 0.0.0.0 --port 4000

# build for production with minification
npm run build
```
